Nodejs Express Test setup
=======================================

Getting Started
---------------

```bash
# Clone the repository
git clone https://github.com/Priyanka1207/node-test-setup.git

# Change directory
cd myapp

# Install dependencies
npm install

# Start it in simple mode
npm start

Models
------
Use of sequelizer database tables schema validator

Environment Variables
---------------------
# Application Configuration
APP_NAME=your app name
APP_ENV=local
APP_PORT=3000

# Database Configuration
DB_HOST=local
DB_USER=user name
DB_PASS= password
DB_NAME=Name of database

# Token
SECRET_TOKEN=your secret key


Tests
-----
User CRUD operations with JWT authentiacation and email notification.

Requirements
------------
* [Nodejs](https://nodejs.org/en/) >= 7.6
* [Mysql]
